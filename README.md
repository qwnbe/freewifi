# SpermSound

SpermSound — это NFC метка, которая включает стоны, музыку и еще много чего другого. Работает в 10 раз дешевле, чем Soundtag.

## Особенности

- **Доступность:** В 10 раз дешевле
- **Легкость в использовании:** Простая документация
